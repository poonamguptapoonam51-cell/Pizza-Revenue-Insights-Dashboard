# 🍕 Pizza Revenue Insights Dashboard

## 📊 Project Overview
This project analyzes pizza sales data to identify trends, customer behavior, and revenue insights using Excel / Power BI.

## 🔧 Tools Used
- Microsoft Excel
- Power BI

## 📌 Key Insights
- Higher sales observed on weekends
- Consistent pricing across products
- Peak ordering during evening hours

## 📈 Dashboard Features
- Total Revenue & Orders
- Sales by Month
- Sales by Day
- Top Selling Pizza

## 🧹 Data Cleaning
- Converted raw transposed data into structured tabular format
- Removed duplicates and unnecessary columns
